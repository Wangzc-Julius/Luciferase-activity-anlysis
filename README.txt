
# SARS-CoV-2 Replicon Inhibition Analysis Report

This folder contains an interactive HTML report for SARS-CoV-2 replicon inhibition screening.

## How to Use:

1. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
2. No internet connection required - all files are self-contained
3. Click on any image to view it in full size
4. Use the search box to find specific mini-proteins
5. Use filter buttons to view specific groups

## Contents:

- `index.html` - Main interactive report
- `plots/` - All individual analysis plots
- `data/` - Processed data files

## Sharing Instructions:

To share this analysis with colleagues:
1. Zip this entire folder
2. Send the ZIP file
3. Recipients can unzip and open `index.html`

## Analysis Details:

- Total groups analyzed: 72
- Time points: 0, 6, 12, 24 hours
- Technical replicates: 3-4 per group
- Controls included: Mock, WT+N, Remdesivir
